# Fernanda-Agrinho-2026
Este portal educativo, desenvolvido para o Concurso Agrinho 2026, oferece uma experiência interativa sobre sustentabilidade e tecnologia agrícola, baseada em dados da Embrapa e da FAO. O site integra funcionalidades de acessibilidade, tradução e um quiz didático, promovendo o pensamento crítico sobre inovação e responsabilidade ecológica no campo.
